# 四柱推命における格局（気質タイプ）を求めるアルゴリズム

## 概要

四柱推命において、格局（Kakukyoku/性格タイプ）は日干（Day Master）を中心に、その人の天干と地支の組み合わせによって決定される重要な要素です。格局は「身強（強い日干）」と「身弱（弱い日干）」の大きく2つに分類され、さらにその中で10種類の格局タイプに細分化されます。本ドキュメントでは格局を決定するアルゴリズムの詳細を記述します。

## 格局を求める基本手順

1. 蔵干（地支に内蔵された十干）を求める
2. 天干と地支の通変星を求める
3. 極身強か極身弱かを判定する
   - 該当する場合は特別格局を求める
4. 極身強にも極身弱にも該当しない場合は、普通格局を求める
5. 普通格局の場合、身強か身弱かを判定する
6. 通変星の数で格局の種類が決まる

## 詳細アルゴリズム

### 1. 蔵干（地支に内蔵された十干）を求める

蔵干早見表を使用して、命式の「年柱」「月柱」「日柱」「時柱」の各地支に対応する蔵干を調べる。蔵干には「本気」「中気」「余気」の3つがあり、場合によっては中気がないこともある。

#### 例：1985年5月8日06時31分（時差修正後）生まれの人の場合
1. 命式を「年柱：乙丑、月柱：辛巳、日柱：丁未、時柱：癸卯」と確認
2. 年支の「丑」の蔵干：本気「己」、中気「辛」、余気「癸」
3. 月支の「巳」の蔵干：本気「丙」、中気「庚」、余気「戊」
4. 日支の「未」の蔵干：本気「己」、中気「丁」、余気「乙」
5. 時支の「卯」の蔵干：本気「乙」、中気なし、余気「甲」

### 2. 天干と地支の通変星を求める

通変星早見表を使用して、命式の日干と他の天干・地支の蔵干との関係を調べる。通変星は以下の10種類がある：

| 通変星の種類 | 説明 |
|------------|------|
| 比肩（ひけん） | 日干と同質で陰陽が同じ |
| 劫財（ごうざい） | 日干と同質で陰陽が違う |
| 食神（しょくしん） | 日干が生じるもので陰陽が同じ |
| 傷官（しょうかん） | 日干が生じるもので陰陽が違う |
| 偏財（へんざい） | 日干が剋すもので陰陽が同じ |
| 正財（せいざい） | 日干が剋すもので陰陽が違う |
| 偏官（へんかん） | 日干を剋すもので陰陽が同じ |
| 正官（せいかん） | 日干を剋すもので陰陽が違う |
| 偏印（へんいん） | 日干を生じるもので陰陽が同じ |
| 印緑（いんりょく） | 日干を生じるもので陰陽が違う |

#### 例：日干が「丁」（火）の場合の通変星
- 丁との関係で、比肩は「丁」、劫財は「丙」など
- 通変星早見表を使って、命式の天干および地支の蔵干それぞれについて通変星を確認

### 3. 極身強か極身弱かを判定する

以下の条件に基づいて極身強か極身弱かを判定する：

#### 極身強の条件：
- 条件A：得令（当旺、または次旺）の関係の十二支がある
- 条件B：日干を強める地支が2つ以上ある
- 条件C：日干を強める天干が2つ以上ある

#### 極身弱の条件：
- 条件A：失令、または月支に日干を弱める十二支がある
- 条件B：日干を弱める地支が3つ以上ある
- 条件C：日干を弱める天干が2つ以上ある

### 4. 特別格局と普通格局の判定

#### 特別格局の判定プロセス
1. まず条件A〜Cに基づいて極身強または極身弱であることを確認する
2. 命式全体（年・月・日・時の天干と地支の蔵干すべて）の通変星の数を数える
3. 特定の通変星の組み合わせが多いかどうかで特別格局のタイプを判定する

#### 特別格局の種類と特徴

##### 極身強の特別格局：
1. **従旺格**（じゅうおうかく）
   - 特徴：主体性があり、自分の思った通りに人生を突き進む
   - 通変星：命式に比肩と劫財が多い
   - 比肩：日干と同じ天干（陰陽も同じ）
   - 劫財：日干と同じ天干（陰陽が異なる）
   - 例：日干が甲の場合、比肩は甲、劫財は乙

2. **従強格**（じゅうきょうかく）
   - 特徴：独自の人生観を持ち、学識の充実した人生を歩む
   - 通変星：命式に偏印と印緑が多い
   - 偏印：日干を生む天干（陰陽も同じ）
   - 印緑：日干を生む天干（陰陽が異なる）
   - 例：日干が甲（木）の場合、偏印は癸（水）、印緑は壬（水）

##### 極身弱の特別格局：
1. **従児格**（じゅうじかく）
   - 特徴：社交的で頭の回転が速く、鋭い感性と人生観を持つ
   - 通変星：命式に食神と傷官が多い
   - 食神：日干が生じるもの（陰陽が同じ）
   - 傷官：日干が生じるもの（陰陽が違う）
   - 例：日干が甲（木）の場合、食神は丙（火）、傷官は丁（火）

2. **従財格**（じゅうざいかく）
   - 特徴：とても強い財運を持ち、人間関係にも恵まれる
   - 通変星：命式に偏財と正財が多い
   - 偏財：日干が剋すもので陰陽が同じ
   - 正財：日干が剋すもので陰陽が違う
   - 例：日干が甲（木）の場合、偏財は戊（土）、正財は己（土）

3. **従殺格**（じゅうさつかく）
   - 特徴：忍耐強く封建的な世界を好み、目上につき従う
   - 通変星：命式に偏官と正官が多い
   - 偏官：日干を剋すもので陰陽が同じ
   - 正官：日干を剋すもので陰陽が違う
   - 例：日干が甲（木）の場合、偏官は庚（金）、正官は辛（金）

4. **従勢格**（じゅうせいかく）
   - 特徴：円満な性格で、環境や状況に柔軟に対応していく
   - 通変星：命式に食神、傷官、偏財、正財、偏官、正官が均等にある
   - これらの6種類の通変星がバランスよく存在する場合

#### 「多い」の判定について
- 通変星の数を数える際は、年・月・日・時の天干と地支の蔵干すべてを対象とする
- 特定の通変星（または組み合わせ）が他と比較して明らかに多い場合、その特別格局と判定
- 例えば、比肩と劫財の合計が全体の30%以上を占めるなど
- 診断結果は別冊の「個人データ表」に記入して確認することが推奨される

#### 特別格局の割合
- 格局全体のうち、約16タイプ（特別格局6タイプ、普通格局10タイプ）ある
- 特別格局は全体の約15%の人が該当し、残りの約85%が普通格局に該当する
- 身強・身弱のポイントで診断すると、特別格局にあたる人は極身強か極身弱となる

### 5. 普通格局の判定と優先順位

普通格局は以下の優先順位で判断する：

1. 「建禄格」になる日干と月支の組み合わせがあるか調べる
2. 「月刃格」になる日干と月支の組み合わせがあるか調べる
3. 月支の蔵干と同じ五行の十干が、月干にあるかを調べる
4. 月支の蔵干と同じ五行の十干が、年干か時干に出ているか調べる
   - 複数出ていたら、月支の蔵干の本気、中気、余気の順で優先する
5. 上記の条件に当てはまらない場合、「月支蔵干深浅表」を使って格局を求める

### 普通格局の種類
- 建禄格：独立心が強く負けず嫌いな性格で、人生を切り拓く
- 月刃格：プライドが高く、自分の世界や価値観を重視する
- 食神格：鋭い感覚を持ち快楽主義者で、のびのびと生きる
- 傷官格：独自の感性の持ち主で、専門技術の習得にも長ける
- 偏財格：社交的で義理人情に厚く、物質生活を重んじる
- 正財格：現実を合理主義者で、現実的な価値判断をする
- 偏官格：正義感にあふれ、強い者を抑えて弱い者を助ける
- 正官格：まじめで大言を表に出さず、家に規律や礼儀を重んじる
- 偏印格：知的好奇心が旺盛で、内面世界の探求を重視する
- 印緑格：甘えん坊が、好奇心を満たすことに喜びを感じる

## 身強・身弱の判定条件

身強・身弱を判定するための条件は以下の通り：

A) 得令（当旺・次旺）にあてはまる
B) 地支が日干を強めている
C) 天干が日干を強めている 
D) 地支に日干の五行を強める三方会局が形成されている
E) 地支に日干の五行を強める三合会局が形成されている
F) 日干が「成」か「己」で、地支に四墓土局が形成されている

## 実装上の注意点

- 正確な格局判定には、節入日と時間の詳細な計算が必要
- 蔵干早見表、通変星早見表、月支蔵干深浅表などのデータベースが必要
- 複数の条件を組み合わせて判定するロジックが必要
- 日干の強弱は相対的なもので、0ポイントに近い「中和」の状態もある

## 結論

格局の判定は四柱推命における重要な解析ステップであり、命式全体のバランスと日干の強弱に基づいて行われる。この格局判定によって、個人の気質的特徴や人生の指針を読み解くことができる。
